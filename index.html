Algorithm Illustrator

Algorithm Illustrator is an interactive visualization site for foundational algorithms. It helps you connect algorithm theory to actual execution by showing every state transition step-by-step.

Each model page uses a step engine: the algorithm emits discrete events, and the UI renders those events with play, pause, step, back, and speed controls.

Quick Tutorial

  1. Open a model page (Search, Sort, or Greedy).
  2. Select an algorithm from the tabs or dropdown.
  3. Read the concept/details on the left.
  4. Click `Build Steps`, then use `Step` or `Play`.
  5. Use `Back`, `Pause`, and speed to inspect behavior carefully.

Visual Algorithm Explorer

Explore search, sorting, and greedy algorithms with step-by-step playback controls. Each model page includes a state panel so you can trace exactly how each algorithm evolves over time.
